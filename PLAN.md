**Fase 1: MVP - Demo Zacapa.**
*   **Alcance:** Nivel 1-8. Pueblo Santiago, mecánicas base (agricultura, caza, recolección), 5 misiones, 5 enemigos, 1 jefe.
*   **Técnica:** Crear CLI base (Node.js) para generar JSON de esta región.
*   **Meta:** Demo jugable autocontenida.

**Fase 2: Expansión a Oriente.**
*   **Alcance:** Regiones Chiquimula, Izabal, El Progreso. Nuevos jefes, misiones, mecánicas de comercio.
*   **Técnica:** Ampliar CLI para soportar múltiples regiones y biomas.
*   **Meta:** Probar generación procedural y viajes largos.

**Fase 3: Núcleo Narrativo.**
*   **Alcance:** Regiones del centro, incluyendo Guatemala. Introducción de Karen Sayle. Magia natural.
*   **Técnica:** CLI genera diálogos y eventos de historia principal.
*   **Meta:** Establecer la trama central.

**Fase 4: Sistemas Avanzados y Altiplano.**
*   **Alcance:** Regiones del altiplano. Crafteo, alquimia, herrería. Monturas.
*   **Técnica:** CLI genera árboles de habilidades complejos y recetas.
*   **Meta:** Profundidad de juego.

**Fase 5: Contenido Final.**
*   **Alcance:** Regiones finales (Verapaces, Huehuetenango, Petén). Bestias míticas, equipo legendario.
*   **Técnica:** CLI genera jefe final procedural y finales alternativos.
*   **Meta:** Completar el juego.
