Crónicas de Q'anil - Elementos UI

Este paquete contiene UI sprites recortados para:
- Menú de inicio
- Menú de pausa

Importante: estos elementos son sprites de interfaz/HUD, no tiles del mapa.
Úsalos como Image/Sprite/TextureRect/Button según tu motor.

Incluye:
- source_sheets/: hojas limpias completas
- start_menu/: logo, botones, paneles, placas y ornamentos
- pause_menu/: header, botones, paneles, iconos, placas y ornamentos
- manifest.json: listado de archivos y tamaños
